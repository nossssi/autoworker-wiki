▶ 오토워커 작동 플로우는 어떻게 되나요? 어디까지 자동이고 어디서 수동 개입이 필요한가요?
① 대본 만들기 → 클로드 스킬로 자동화 (자동)
② 튜브워커에 대본 넣기 → 이미지 선택 등은 수동으로 조정 가능 (반자동)
③ CapCut 결과물 확인 → 다운로드 후 동영상 추출, 세부사항 편집이 필요하면 수동 편집 (수동)
④ 유튜브 업로드 → 명령어로 업로드 자동화 (자동)

즉, 대본 생성과 업로드는 자동이고, 중간의 이미지 선택과 최종 편집 단계에서 사용자가 원하는 대로 조정할 수 있습니다.
